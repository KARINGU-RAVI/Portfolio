# Portfolio

This folder contains the main frontend (and optional backend) for the portfolio application. Everything lives inside the `portfolio` directory.

## Setup

```bash
cd portfolio
npm install
```

## Frontend commands

- `npm run dev` (or `npm start`) – start the Vite development server (hot reload)
- `npm run client:dev` – same as above
- `npm run build` – build the frontend for production (alias for `client:build`)
- `npm run client:build` – build the frontend for production
- `npm run client:preview` – preview the production build locally
